Green Solvent for Metal-Organic Framework Synthesis

Abstract:

Metal-Organic Frameworks (MOFs), which can be used for the adsorption of carbon dioxide in order to combat pollution, are expensive to synthesize. One way to reduce the cost of producing MOFs is utilizing organic solvents with lower boiling points than those typically used in MOF synthesis. 2-Methyl Tetrahydrofuran (2-MeTHF) is one example of a solvent that could be used in place of traditional solvents such as Dimethylformamide (DMF) and N,N-Dimethylacetamide (DMA). 2-MeTHF has a lower boiling point than that of DMF and DMA, which requires lower temperatures and reaction times during synthesis and degassing, thus making the process more energy and cost efficient. The purpose of this work was to determine the feasibility of using 2-MeTHF in MOF synthesis by comparing the surface area, porosity, crystallinity and stability of resulting MOFs to those prepared using traditional solvents. Three different MOFs, HKUST-1, Cu(BDC) and Mg(BDC), were prepared in 2-MeTHF. The results of this study show that 2-MeTHF yields MOFs with high crystallinity and surface areas at lower reaction temperatures and shorter reaction times, resulting in increased energy and cost efficiency. From these results, it can be concluded that 2-MeTHF may be a viable solution to lowering costs of the production of MOFs and increasing their potential in clean energy applications.

Hangman Game

Language:

Python 3

Code:

            #HANGMAN
            """Note: if same letter guessed twice, that turn is used up"""
            print("Welcome to Hangman!")
            repeat = "y"

            while (repeat == "y"):

              #user prompt
              word = (input("Pick a word: ")).lower()
              max_guesses = int(input("Enter the maximum number of guesses: "))
              blank = ["_ "] * len(word)
              print (" ".join(blank))
              print (" ")
              wrong_letters = []
              correct_letters = []
              correct = 0
              guesses = 0
              wrong = 0

              #guess
              for x in range(max_guesses):
                guesses += 1
                letter = input("Guess # " + str(guesses) + ". Guess a letter: ")

                #check if letter already used
                if letter in (correct_letters or wrong_letters):
                  print ("You already used that letter!\n")
                else:
                  #check if letter in word
                  for i in range(len(word)):
                    if letter == word[i]:
                      blank[i] = letter
                      correct_letters.append(letter)
                      correct +=1

                  #if letter not in word
                  if letter not in word:
                    print("Letter not in word. Try Again.")
                    wrong += 1
                    wrong_letters.append(letter)

                  #print guess so far
                  print (" ")
                  print (" ".join(blank))
                  print ("Correct: " + str(correct))
                  print ("Wrong: " + str(wrong))
                  print ( "Wrong letters: " + " ".join(wrong_letters))
                  print (" ")

                  #if correct answer before max guesses
                  if (correct == len(word)):
                    break

              #game result
              if (correct == len(word)):
                print ("Congratulations. You Win!")
              elif guesses == max_guesses:
                print ("Game Over. Number of maximum guesses reached. You Lose!")
                print ("Answer: " + word)
              else:
                print ("Error!")

              repeat = (input("Would you like to play again? Enter Y to play again or N to exit --> ")).lower()
